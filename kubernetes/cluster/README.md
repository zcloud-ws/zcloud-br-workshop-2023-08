# Pré requisitos para criar o cluster

- Ter acesso root SSH a um servidor (VM ou Bare Metal);
- Configurar conexão por chave SSH no servidor (recomendado) 
- Liberar acesso às portas TCP: 22, 80, 443, 6443, 10250 (metrics)
